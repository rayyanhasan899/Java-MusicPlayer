# Java Terminal Music Player

A console-based music player developed in Java that allows users to manage playlists and play songs through a terminal interface.

---

## Features

- Song selection from local storage
- Playlist creation and management
- Basic playback controls
- File handling for music directories
- Modular Java architecture

---

## Technologies Used

- Java
- Object-Oriented Programming
- File Handling
- Terminal-based UI

---

## Project Structure

- **Menu** – Handles user interaction
- **MusicPlayer** – Manages playback and song selection
- **PlaylistManager** – Handles playlists
- **FileHandler** – Reads music files from directories
- **Main** – Program entry point
